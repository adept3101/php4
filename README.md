# Руководство пользователя к использованию программы.

Для использования веб-приложения необходимо зарегестрироваться, а затем войти в аккаунт.


Пример:

```
login
pass
```

После входа вы попадает на главную страницу. На этой странице находится вся информация о камерах.
Вы также можете добавить камеру в таблицу самостоятельно используя блок справа от таблицы, введя необходимые данные.


Пример:

```
15
Panasonic ultra
Tiny
Grey
3000
Металл
Japan
```

В таблице можно удобно сортировать желаемые данные кликнув по ним левой клавишой мыши.


Снизу находится информация о пользователе: логин, дата регистрации и id.


Сверху есть кнопка "Выйти" нажав на нее вы попадете на окно авторизации.
